# basic command
> git 기본 문법 정리

## 초기 설정
- git을 설치 후 한번만 실행
```bash
git config --global user.email <email>
git config --gloobal user.name <name>
```

## git 저장소 만들기

- `git init` 
    - `.git directory`를 생성해주는 명령어 / 관리하고 싶은 최상위 위치에서 실행

## 코드 수정하고 저장소에 저장하기

- `status`
    - `git status` : 현재 git으로 관리되고있는 파일/폴더의 상태를 출력
- `add`
    - `git add`
    - `git add . `
